# 高拟态模式共同标准

这份标准只回答一件事：

> **这篇文章像不像指定作者本人，而不是只像一个强写手。**

## 一票否决项

命中任一条，直接判定不通过：

1. 只学到口头禅，没学到论证动作
2. 句子局部像，整篇节奏不像
3. 作者世界观被通用素材库冲淡
4. 主题明显超出作者带宽，还硬装像
5. 读起来像 cosplay，不像本人

## 六项核心标准

### 1. 开头动作像

- 是否像该作者习惯的开场方式
- 是从趋势、痛点、判断、愿景，还是从别的入口进

### 2. 判断结构像

- 该作者通常怎样下判断
- 是先重定义概念，还是先打破误区，还是先立新框架

### 3. 推进节奏像

- 每一段是怎样往前走的
- 是层层抽象，还是框架压缩，还是概念驱动

### 4. 世界观像

- 文章的价值排序是否像这个作者
- 他最看重什么、最反感什么，是否写出来了

### 5. 句法和意象像

- 不是抄原句
- 而是像他惯用的比喻、命名方式、对比方式、压缩方式

### 6. 收束方式像

- 结尾是否像该作者会给出的出口
- 是判断、使命、行动、愿景，还是别的方式

## 难伪造信号优先级

高拟态判断时，优先看这些“更难靠公共词表伪装”的信号：

1. **开头动作**：是怎样开，不只是开头用了什么词
2. **重定义路径**：作者如何把旧问题换一个坐标系解释
3. **抽象抬升方式**：他如何从战术问题抬到身份、系统、主权或长期结构
4. **价值排序**：他到底更看重什么，不看重什么
5. **收束方式**：最后把读者带向判断、愿景还是行动入口

这些信号比“有没有写系统 / 地图 / 复利”更重要。

## 动作链要求

高拟态模式不要只在正文里散落作者信号。

至少要出现一条完整动作链，例如：

1. **Why now -> 旧默认失效 -> 新默认建立**
2. **重定义问题 -> 把战术问题抬到身份/系统层 -> 用更大的方向收束**

如果正文只有局部句法像、关键词像，但没有完整动作链，那它依然更像表层模仿，而不是高拟态。

## 公共词信号降权

下面这些词或意象，本身不构成强高拟态证据：

- 系统
- 地图
- 拼图
- 主权
- 复利
- 身份升级

原因很简单：

> 这些已经太像 Dan Koe 的公共模仿词表，baseline 也很容易手工逼近。

如果一篇稿子只有这些公共信号，却没有作者动作组合、重定义路径和价值排序，那它更像 cosplay，而不像本人。

## 降级规则（强制）

如果主题超出作者原生议题带宽：

1. 明确提醒
2. 自动降级到强文章模式
3. 不继续输出伪拟态稿

## 通过标准

至少同时满足：

1. 开头动作像
2. 判断结构像
3. 推进节奏像
4. 世界观像
5. 收束方式像

句法和意象像是加分项，但不能代替前五项。
公共词信号只能当辅证，不能当主证。
完整动作链比零散信号更重要。

## 推荐验证方式

用未参与构建 profile 的作者原文做盲比。

判断重点不是“好不好”，而是：

- 这像不像这个作者本人会写出来的东西
- 如果去掉作者名字，我还能不能从文风和推进里认出他
