# Profile Probe｜评分模板

## 评分对象

主评分对象：`response.md`

可参考：

- `high-likeness-rubric.md`
- `<author-id>/acceptance-checklist.md`

默认不要先读 `run_summary.md`，避免污染判断。

## 推荐维度（100 分）

### A. 标题 / 开头抓力（20）

### B. 正文推进动作（20）

### C. 大众翻译能力（15）

### D. 现实收益落点（15）

### E. 内容 -> 信任 -> 产品意识（10）

### F. 收束传播力（10）

### G. 反 cosplay 稳定性（10）

## 必要检查

### 一票否决

- 只学到口头禅，没学到论证动作
- 句子局部像，整篇节奏不像
- 作者世界观被通用素材库冲淡
- 主题超带宽还硬装像
- 读起来像 cosplay，不像本人

### acceptance checklist

- 逐项勾选
- 不允许跳过

### false-positive 检查

- 题材是否天然抬分
- 关键词是否过度抬分
- 去掉强信号后，动作链是否仍成立

## 输出要求

必须包含：

- 各维度分数
- 总分
- 是否 >= 90
- 通过 / 不通过结论
- 下一轮 1–3 个修复目标
